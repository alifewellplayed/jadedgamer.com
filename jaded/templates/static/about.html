Jadedgamer is a website aggregator written by Tyler Rilling. It grabs feeds from the most popular in the industry, allowing you to quickly digest the latest news. The feed list is currently currated and maintained by myself, with the ability to submit feeds within the coming soon.

Server Software

This site is written in Python, using the Django web framework. The site is built on a serverless architecture and hosted on various Amazon Web Services, primarily, Lambda and CloudFront. Statistics are collected using Mixpanel and Google Analytics.

Feature Requests

This site is in active development, with new features being added constantly. If there's a particular feature you'd like to see however, feel free to open an issue on Github, or email me.

You can also follow along with it's development on my blog, A Life Well Played.
